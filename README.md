# TimeBound

[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)

TimeBound is a premium study timer focused on time discipline and deep focus. It helps users structure their study sessions using the Pomodoro technique, enforce time boundaries, and build consistency with a calm, elegant dark UI and optional cloud sync.

## About
TimeBound is designed for users who want structure, not motivation. Instead of flashy productivity tricks, it focuses on disciplined time management through clearly defined focus and break sessions. With task tracking, session statistics, and optional Firebase sync, TimeBound supports consistent, distraction-free study habits across devices.

---

## Features
- Pomodoro-based focus, short break, and long break sessions  
- Customizable timer durations  
- Elegant circular progress indicator with smooth animations  
- Task management for daily goals  
- Session statistics, total focus time, and streak tracking  
- Firebase authentication with cloud sync (optional)  
- Offline support with automatic data persistence  
- Keyboard shortcuts for fast control  
- Responsive, dark, premium UI  
- PWA support for mobile installation
